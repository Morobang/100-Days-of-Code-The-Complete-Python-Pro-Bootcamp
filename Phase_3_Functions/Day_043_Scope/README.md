# Day 43: Scope

**Phase:** 3 — Functions
**Difficulty:** ⭐⭐

## What you'll learn
- **Local scope** — variables that live only inside a function
- **Global scope** — variables defined at the module level
- The **LEGB rule** — how Python resolves names (Local → Enclosing → Global → Built-in)
- The `global` keyword — how to modify a global variable inside a function
- Why using `global` is usually a design smell
- Reading a global without `global` — allowed; modifying without it — error
