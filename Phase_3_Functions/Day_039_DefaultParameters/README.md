# Day 39: Default Parameters

**Phase:** 3 — Functions
**Difficulty:** ⭐⭐

## What you'll learn
- Giving parameters a default value: `def greet(name, greeting='Hello'):`
- Calling with or without the optional argument
- Why defaults must come after required parameters
- The mutable default trap — never use `[]` or `{}` as a default
- Mixing required and optional parameters
