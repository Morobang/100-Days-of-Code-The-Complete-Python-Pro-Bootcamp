# Phase 3 — Functions

**Days 36–48 | 2 projects**

Functions are how you organize code so it can be reused. Instead of writing the same logic over and over, you define it once and call it whenever you need it. This phase covers everything from basic function definitions to advanced features like `*args`, `**kwargs`, and lambda functions.

---

## What you will learn

- Defining and calling functions
- Passing data into functions (parameters and arguments)
- Getting data back from functions (return values)
- Flexible functions with default parameters, `*args`, and `**kwargs`
- Variable scope — where your variables can be seen
- Lambda functions and functional tools like `map()` and `filter()`
