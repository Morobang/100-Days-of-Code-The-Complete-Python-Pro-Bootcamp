# Day 47: Map and Filter

**Phase:** 3 — Functions
**Difficulty:** ⭐⭐

## What you'll learn
- `map(func, iterable)` — apply a function to every item in an iterable
- `filter(func, iterable)` — keep only items where the function returns `True`
- Both return **iterators** — wrap with `list()` to materialise results
- Using named functions and lambdas with `map` and `filter`
- How `map`/`filter` compare to list comprehensions — when each is cleaner
