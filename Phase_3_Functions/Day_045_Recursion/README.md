# Day 45: Recursion

**Phase:** 3 — Functions
**Difficulty:** ⭐⭐⭐

## What you'll learn
- The two requirements of every recursive function: **base case** and **recursive case**
- How the **call stack** grows and shrinks with each recursive call
- Classic recursive patterns: countdown, factorial, sum of a list
- Recursion vs iteration — when each is clearer
- Python's recursion depth limit and `RecursionError`
