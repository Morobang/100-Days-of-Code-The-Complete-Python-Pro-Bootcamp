# Day 40: *args — Variable Positional Arguments

**Phase:** 3 — Functions
**Difficulty:** ⭐⭐

## What you'll learn
- `*args` lets a function accept **any number of positional arguments**
- Inside the function, `args` is a **tuple**
- Iterating over `args` with a `for` loop
- Combining `*args` with regular parameters (order rules)
- Unpacking a list/tuple into positional arguments with `*`
