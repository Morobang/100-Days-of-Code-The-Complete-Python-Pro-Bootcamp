# Day 38: Return Values

**Phase:** 3 — Functions
**Difficulty:** ⭐⭐

## What you'll learn
- The `return` statement — sending a value back to the caller
- Storing and using return values
- Returning different types (numbers, strings, booleans, lists)
- Returning **multiple values** (received as a tuple)
- Early return to exit a function before the end
- `None` — what you get when there is no `return`
