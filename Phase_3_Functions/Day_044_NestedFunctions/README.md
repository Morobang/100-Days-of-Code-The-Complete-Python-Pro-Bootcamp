# Day 44: Nested Functions

**Phase:** 3 — Functions
**Difficulty:** ⭐⭐⭐

## What you'll learn
- Defining a function **inside** another function
- **Enclosing scope** — the "E" in LEGB, now in action
- **Closures** — inner functions that remember variables from their enclosing scope
- **Factory functions** — functions that create and return other functions
- The `nonlocal` keyword — modifying a variable in the enclosing scope
