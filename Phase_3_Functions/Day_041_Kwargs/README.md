# Day 41: **kwargs — Variable Keyword Arguments

**Phase:** 3 — Functions
**Difficulty:** ⭐⭐

## What you'll learn
- `**kwargs` lets a function accept **any number of keyword arguments**
- Inside the function, `kwargs` is a **dict**
- Iterating over `kwargs.items()`
- Combining `**kwargs` with regular params and `*args` (order rules)
- Unpacking a dict into keyword arguments with `**`
- Practical uses: flexible config, building HTML/templates
