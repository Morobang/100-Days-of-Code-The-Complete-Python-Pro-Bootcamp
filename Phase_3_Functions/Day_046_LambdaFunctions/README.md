# Day 46: Lambda Functions

**Phase:** 3 — Functions
**Difficulty:** ⭐⭐

## What you'll learn
- The `lambda` syntax — anonymous one-expression functions
- Using lambda as a `key=` argument for `sorted()`, `.sort()`, `min()`, `max()`
- Lambda with multiple parameters
- When to use lambda vs a named `def`
- What lambda **cannot** do (multi-line, statements, assignments)
