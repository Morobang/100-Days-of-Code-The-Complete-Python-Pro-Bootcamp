# Day 51: Working with Paths

**Phase:** 4 — Files and Errors
**Difficulty:** ⭐⭐

## What you'll learn
- Absolute vs relative paths
- `os.getcwd()` — where Python is running from
- `os.path.join()` — building paths that work on every OS
- `os.path.exists()`, `os.path.isfile()`, `os.path.isdir()`
- `os.listdir()` — listing directory contents
- `os.path.basename()`, `os.path.dirname()`, `os.path.splitext()`
- A first look at `pathlib.Path` — the modern alternative
