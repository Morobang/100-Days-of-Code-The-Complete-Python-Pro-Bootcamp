# Day 50: Writing Files

**Phase:** 4 — Files and Errors
**Difficulty:** ⭐⭐

## What you'll learn
- Write mode `'w'` — creates a new file or **overwrites** an existing one
- Append mode `'a'` — adds to the end of an existing file
- `.write(string)` — write a string (no automatic newline)
- `.writelines(list)` — write each string in a list
- Always use `with` to guarantee the file is flushed and closed
