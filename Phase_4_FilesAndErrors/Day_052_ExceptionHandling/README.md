# Day 52: Exception Handling

**Phase:** 4 — Files and Errors
**Difficulty:** ⭐⭐

## What you'll learn
- What exceptions are and when Python raises them
- The `try / except` block — catching errors gracefully
- Catching specific exception types (`ValueError`, `ZeroDivisionError`, `FileNotFoundError`, …)
- `except ExceptionType as e:` — accessing the error message
- Multiple `except` blocks
- The `else` clause — runs only when no exception occurred
- The `finally` clause — always runs, used for clean-up
