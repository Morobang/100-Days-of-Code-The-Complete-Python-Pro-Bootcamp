# Day 49: Reading Files

## What you'll learn
- `open()` — opening a file and the importance of closing it
- The `with` statement — the correct, safe way to open files
- Reading methods: `.read()`, `.readlines()`, `.readline()`, and `for line in f`
- Stripping trailing newlines with `.strip()`
- File encoding and why `encoding='utf-8'` matters
