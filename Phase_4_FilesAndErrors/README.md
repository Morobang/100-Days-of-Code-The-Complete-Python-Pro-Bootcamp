# Phase 4 — Files & Error Handling

**Days 49–56 | 1 project**

Programs that only run in memory lose everything when they close. This phase teaches you to read and write real files, work with common formats like CSV and JSON, and handle errors gracefully so your programs don't crash on unexpected input.

---

## What you will learn

- Reading from and writing to text files
- Working with CSV files (spreadsheet-style data)
- Working with JSON files (structured data from APIs and configs)
- Understanding Python's error types
- Using `try`, `except`, and `finally` to handle errors
- Raising your own exceptions when something goes wrong

## Prerequisites

Phases 1–3 complete (Days 1–48)

- Load and save structured data in CSV and JSON format
- Write programs that handle bad input without crashing
- Give meaningful error messages to users
