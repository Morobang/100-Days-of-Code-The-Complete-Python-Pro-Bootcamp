# Day 49 — Easy Interview Problems

## Speed cheat sheet
| Pattern | Problem type |
|---------|-------------|
| Hash map | Two sum, anagram, duplicate |
| Two pointers | Sorted array problems, palindrome |
| Sliding window | Subarray/substring with property |
| Stack | Brackets, next greater element |
| DP | Counting ways, min/max cost |
| Sorting | Meeting rooms, merge intervals |

## Edge cases to always check
- Empty input
- Single element
- All same elements
- Negative numbers
- Integer overflow

## Interview tips
- Think aloud — say what pattern you're using
- Start with brute force, then optimise
- Test your solution with the provided examples before submitting
