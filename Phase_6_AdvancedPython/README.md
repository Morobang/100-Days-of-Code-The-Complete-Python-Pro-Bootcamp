# Phase 6 — Advanced Python

**Days 71–83 | 2 projects**

This phase covers the parts of Python that separate beginners from developers who write clean, professional code. Generators, decorators, closures, regular expressions, and the standard library — these are tools you will use for the rest of your Python career.

---

## What you will learn

- Generators and `yield` — producing values lazily
- Iterators — understanding how `for` loops actually work
- Decorators — wrapping functions to add behaviour
- Closures — functions that remember their environment
- Context managers — the `with` statement and resource management
- Regular expressions — powerful pattern matching in text
- Working with modules, packages, and the standard library
- Managing dependencies with virtual environments and pip

## Prerequisites

Phases 1–5 complete (Days 1–70)
