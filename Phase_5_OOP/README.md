# Phase 5 — Object-Oriented Programming

**Days 57–70 | 3 projects**

Object-oriented programming (OOP) is a way of organising code around things — objects — that have their own data and behaviour. It's the foundation of most real-world Python programs. This phase takes you from your first class to inheritance, special methods, and polymorphism.

---

## What you will learn

- Defining classes and creating objects
- Giving objects data with `__init__` and attributes
- Giving objects behaviour with methods
- Sharing data across all instances with class attributes
- Building on existing classes with inheritance
- Overriding and extending inherited behaviour
- Using `super()` to call parent class methods
- Special (dunder) methods like `__str__`, `__len__`, `__repr__`
- Controlling attribute access with properties
- Abstract classes and polymorphism
