# Day 2 — Variables & Naming

Variables are named containers for storing data. Today you learn how to create them, name them properly, and understand what type of data they hold.

## What you will learn

- Creating variables with `=`
- Python naming rules (what is and isn't allowed)
- The `snake_case` convention
- Reassigning variables
- Checking the type of a variable with `type()`
