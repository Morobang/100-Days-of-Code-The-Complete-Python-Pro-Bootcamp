# Day 11 — String Slicing

Slicing lets you extract any part of a string using a start position, a stop position, and an optional step. It is one of the most flexible tools in Python.

## What you will learn

- Basic slicing: `s[start:stop]`
- Step: `s[start:stop:step]`
- Omitting start or stop
- Negative indices in slices
- Reversing a string with `s[::-1]`
