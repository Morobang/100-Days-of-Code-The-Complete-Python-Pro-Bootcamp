# Day 9 — String Methods

Strings are not just containers for text — they come with a built-in toolkit of methods for searching, modifying, and checking content.

## What you will learn

- Case: `.upper()`, `.lower()`, `.title()`
- Whitespace: `.strip()`, `.lstrip()`, `.rstrip()`
- Searching: `.count()`, `.find()`, `.startswith()`, `.endswith()`
- Modifying: `.replace()`
- Splitting and joining: `.split()`, `.join()`
- Checking: `.isdigit()`, `.isalpha()`, `.isalnum()`
