# Day 13 — while Loops

A `while` loop runs a block of code repeatedly as long as a condition is True. It is the foundation of repetition in Python.

## What you will learn

- `while condition:` — the basic loop
- Counting up and counting down
- `break` — exit the loop immediately
- `continue` — skip to the next iteration
- Keeping a running total (accumulator pattern)
- Input validation loops
