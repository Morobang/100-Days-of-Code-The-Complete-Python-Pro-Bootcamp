# Day 4 — Numbers & Arithmetic

Python handles numbers in two main forms: integers (whole numbers) and floats (decimals). Today you learn how to do calculations and use Python's built-in math tools.

## What you will learn

- `int` vs `float` — the two number types
- Arithmetic operators: `+`, `-`, `*`, `/`, `//`, `%`, `**`
- The difference between `/` (true division) and `//` (integer division)
- The modulo operator `%` (remainder)
- Order of operations
- The `math` module for advanced calculations
