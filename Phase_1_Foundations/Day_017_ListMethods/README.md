# Day 17 — List Methods

Lists come with a full set of built-in methods for sorting, searching, and restructuring. Today you learn the most useful ones.

## What you will learn

- `.sort()` vs `sorted()` — in-place vs returning new
- `.reverse()`
- `.count()` and `.index()`
- `.pop()` and `.insert()`
- `.remove()` revisited
- `min()`, `max()`, `sum()` — built-in functions for lists
- `.copy()` — why copying matters
