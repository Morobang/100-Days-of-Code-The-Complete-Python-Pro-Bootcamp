# Day 1 — Printing & Output

Your very first Python program. Today you learn how to make Python display things on the screen.

## What you will learn

- `print()` — the most used function in Python
- Printing multiple values at once
- The `sep` and `end` parameters
- Escape sequences (`\n`, `\t`)
- A first look at f-strings (full lesson on Day 10)
