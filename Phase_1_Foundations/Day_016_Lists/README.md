# Day 16 — Lists

A list is an ordered collection of values. It can hold any mix of types, and you can add, remove, or change items at any time.

## What you will learn

- Creating lists with `[]`
- Indexing and negative indexing
- Slicing lists
- `len()` — number of items
- `append()` and `remove()` — adding and removing items
- Modifying items by index
- The `in` operator for membership checks
- Iterating over a list with `for`
