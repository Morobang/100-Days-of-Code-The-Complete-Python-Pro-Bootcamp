# Day 18 — Tuples and Sets

Two more collection types, each with a specific purpose:

- **Tuple** — ordered, immutable. Use when the data should not change.
- **Set** — unordered, unique values only. Use when duplicates don't make sense.

## What you will learn

- Creating and indexing tuples
- Tuple packing and unpacking
- Creating sets and why duplicates are removed
- Set operations: union, intersection, difference
- Set methods: `add()`, `remove()`, `discard()`
