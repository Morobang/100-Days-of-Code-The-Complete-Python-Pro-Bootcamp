# Day 10 — String Formatting

Today you will master f-strings — Python's most powerful tool for embedding values inside text. You also learn how to control alignment, decimal places, and number display.

## What you will learn

- f-strings: embedding expressions inside strings
- Number formatting: `:.2f`, `:,`, `:.2%`
- Alignment: `:<`, `:>`, `:^` with a fixed width
- Fill characters: `f"{value:0>5}"`
- Triple-quoted strings for multi-line text
