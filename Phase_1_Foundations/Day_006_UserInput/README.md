# Day 6 — User Input

So far your programs have used fixed data. Today you make them interactive — users can type in values and the program responds.

## What you will learn

- `input()` — getting text from the user
- `input()` always returns a string
- Converting user input to numbers with `int()` and `float()`
- Building simple interactive programs
