# Day 19 — Dictionaries

A dictionary stores data as **key-value pairs**. Instead of looking up a value by position (like a list), you look it up by name.

## What you will learn

- Creating dictionaries with `{key: value}`
- Accessing, adding, and updating values
- Deleting keys with `del`
- The `in` operator for key checks
- Iterating with `.keys()`, `.values()`, `.items()`
- Building dictionaries with loops
