# Day 3 — String Basics

A string is any text — a name, a sentence, a single character. Today you learn the fundamentals of working with strings in Python.

## What you will learn

- Creating strings with single and double quotes
- The `len()` function — how long is a string?
- Accessing individual characters by index
- String concatenation (joining strings with `+`)
- String repetition (repeating with `*`)
