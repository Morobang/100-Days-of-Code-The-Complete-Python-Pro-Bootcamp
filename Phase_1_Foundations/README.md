# Phase 1 — Foundations

**Days 1–21 | 3 projects**

This is where everything starts. You will write your first Python programs, learn how to store and work with data, take input from users, make decisions with conditionals, and repeat actions with loops.

No prior experience needed. Day 1 assumes you have never written a single line of code.

---

## What you will learn

- Printing output to the screen
- Storing data in variables
- Working with text (strings) and numbers
- Getting input from the user
- Making decisions with `if`, `elif`, `else`
- Repeating code with `while` and `for` loops

## Prerequisites

None. This is the starting point.
