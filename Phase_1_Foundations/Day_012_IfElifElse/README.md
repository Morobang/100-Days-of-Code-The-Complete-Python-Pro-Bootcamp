# Day 12 — if / elif / else

Until now your programs ran every line every time. Today you learn how to make decisions — running different code depending on a condition.

## What you will learn

- `if` — run code only when a condition is True
- `else` — the fallback when the condition is False
- `elif` — check multiple conditions in order
- Comparison operators: `==`, `!=`, `<`, `>`, `<=`, `>=`
- Logical operators: `and`, `or`, `not`
