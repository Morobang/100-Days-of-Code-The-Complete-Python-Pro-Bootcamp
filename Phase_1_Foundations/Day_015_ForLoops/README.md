# Day 15 — for Loops

A `for` loop repeats a block of code once for each item in a sequence. Unlike `while`, you don't manage a counter yourself — Python handles the iteration for you.

## What you will learn

- `for item in sequence:` — iterating over strings
- `range()` — generating sequences of numbers
- `range(start, stop, step)` — controlling the range
- `break` and `continue` inside for loops
- `enumerate()` — looping with an index
