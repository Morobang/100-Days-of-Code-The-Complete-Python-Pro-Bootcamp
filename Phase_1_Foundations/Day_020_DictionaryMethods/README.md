# Day 20 — Dictionary Methods

Dictionaries have a set of built-in methods that make common operations safer and more expressive. Today you learn the essential ones.

## What you will learn

- `.get()` — safe key lookup with a default
- `.keys()`, `.values()`, `.items()` — views of the dictionary
- `.update()` — merge dictionaries
- `.pop()` — remove and return a value
- `.setdefault()` — set a key only if it doesn't already exist
- Nested dictionaries
