# Day 8 — Type Conversion

You have already used `int()` and `float()` with `input()`. Today you learn exactly how Python converts between types, what is possible, and what causes errors.

## What you will learn

- `int()`, `float()`, `str()`, `bool()` — the four main conversion functions
- What each function accepts and what it produces
- What happens when conversion fails (`ValueError`)
- The difference between implicit and explicit conversion
