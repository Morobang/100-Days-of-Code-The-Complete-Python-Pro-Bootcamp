# Day 5 — Booleans & None

Some data is not a number or text — it's a yes/no answer, or the absence of a value. Today you learn Python's `True`/`False` type and the special value `None`.

## What you will learn

- `True` and `False` — the two boolean values
- The `bool` type and `bool()` function
- What makes a value truthy or falsy
- `None` — Python's way of representing "nothing"
