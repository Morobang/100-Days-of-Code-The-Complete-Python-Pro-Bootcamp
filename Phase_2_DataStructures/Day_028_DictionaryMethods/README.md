# Day 28: Dictionary Methods — Deeper

**Phase:** 2 — Data Structures
**Difficulty:** ⭐⭐

## What you'll learn
- `dict()` constructor: two alternative ways to create a dict
- `dict.fromkeys()` — create a dict with uniform default values
- `.update()` with keyword arguments
- `.pop()` with a default to safely remove keys
- Merging two dicts with `**` unpacking
- Finding the key with the highest or lowest value
