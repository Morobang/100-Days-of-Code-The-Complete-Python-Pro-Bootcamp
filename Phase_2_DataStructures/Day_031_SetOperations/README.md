# Day 31: Set Operations — In-Depth

**Phase:** 2 — Data Structures
**Difficulty:** ⭐⭐

## What you'll learn
- In-place set operators: `|=`, `&=`, `-=`, `^=`
- Their method equivalents: `.update()`, `.intersection_update()`, `.difference_update()`, `.symmetric_difference_update()`
- Operations across three or more sets at once
- Practical patterns: finding overlap across multiple groups, filtering against a blocklist, deduplication while processing
