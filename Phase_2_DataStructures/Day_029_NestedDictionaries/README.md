# Day 29: Nested Dictionaries

**Phase:** 2 — Data Structures
**Difficulty:** ⭐⭐⭐

## What you'll learn
- Creating dicts whose values are other dicts
- Accessing deeply nested values with chained `[]` or safe `.get()`
- Modifying and adding nested keys
- Iterating over nested dicts with two-level loops
- Building nested dicts from user input
- Practical patterns: student records, product catalogues, config objects
