# Day 23: List Methods

**Phase:** 2 — Data Structures
**Difficulty:** ⭐⭐

## What you'll learn
- `.append()` vs `.extend()` — the difference matters
- Removing items: `.remove()`, `.pop()`, `del`
- Sorting with `key=` and `reverse=`
- `sorted()` vs `.sort()` — which returns a new list
- Using a list as a **stack** (LIFO) and a **queue** (FIFO)
- `.clear()`
