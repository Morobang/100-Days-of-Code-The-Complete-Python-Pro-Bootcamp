# Day 22: Lists — Creating and Indexing

**Phase:** 2 — Data Structures
**Difficulty:** ⭐⭐

## What you'll learn
- Multiple ways to create lists: literals, `list()`, `list(range())`, repetition
- Positive and negative indexing patterns
- Slicing with step
- Membership testing with `in` / `not in`
- Built-ins: `len()`, `min()`, `max()`, `sum()`
- The copy trap: aliasing vs `.copy()`
