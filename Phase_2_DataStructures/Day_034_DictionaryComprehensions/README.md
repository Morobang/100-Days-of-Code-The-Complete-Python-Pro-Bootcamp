# Day 34: Dictionary and Set Comprehensions

**Phase:** 2 — Data Structures
**Difficulty:** ⭐⭐

## What you'll learn
- Dictionary comprehension syntax: `{key: value for item in iterable}`
- Filtering with `if`: `{k: v for k, v in d.items() if condition}`
- Transforming an existing dict's values
- Inverting a dict in one line
- Set comprehension syntax: `{expression for item in iterable}`
- Choosing between dict comprehensions, set comprehensions, and list comprehensions
