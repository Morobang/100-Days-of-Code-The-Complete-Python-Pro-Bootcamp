# Phase 2 — Data Structures

**Days 22–35 | 3 projects**

Now that you can write basic programs, it's time to work with collections of data. Real programs deal with lists of things, tables of information, and sets of unique items. This phase teaches you the core data structures Python gives you and when to use each one.

---

## What you will learn

- Lists — ordered collections you can change
- Tuples — ordered collections that stay fixed
- Dictionaries — storing data as key-value pairs
- Sets — collections of unique items
- List and dictionary comprehensions — a cleaner way to build collections

## Prerequisites

Phase 1 complete (Days 1–21)

## By the end of Phase 2 you can

- Store and retrieve collections of data
- Choose the right structure (list, tuple, dict, set) for each situation
