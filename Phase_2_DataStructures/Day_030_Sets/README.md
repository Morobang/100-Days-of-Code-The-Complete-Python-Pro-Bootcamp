# Day 30: Sets — Deeper

**Phase:** 2 — Data Structures
**Difficulty:** ⭐⭐

## What you'll learn
- Creating sets with `set()` from any iterable
- `.add()`, `.discard()` vs `.remove()`, `.update()`
- All four set operations: union, intersection, difference, symmetric difference
- Relationship tests: `.issubset()`, `.issuperset()`, `.isdisjoint()`
- `frozenset` — an immutable set (can be used as a dict key)
- Why sets are faster than lists for membership testing
