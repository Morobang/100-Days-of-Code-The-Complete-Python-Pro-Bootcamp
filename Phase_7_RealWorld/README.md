# Phase 7 — Python in the Real World

**Days 84–100 | 5 projects + Final Capstone**

The final phase puts everything together and connects Python to the outside world. You will fetch data from the internet, store it in databases, build command-line tools, write tests, and work with data libraries. Day 100 is your capstone — build something real from scratch.

---

## What you will learn

- Fetching data from the web with `requests`
- Working with REST APIs and JSON responses
- Web scraping with BeautifulSoup
- Storing and querying data with SQLite
- Building command-line apps with `argparse`
- Writing tests with `pytest`
- Type hints and dataclasses for cleaner code
- NumPy, Pandas, and Matplotlib for data work

## Prerequisites

Phases 1–6 complete (Days 1–83)
